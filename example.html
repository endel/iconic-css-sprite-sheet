Iconic CSS Sprite Sheet by Endel Dreyer

A CSS and PNG Sprite Sheet generator for Iconic iconset

Requirements

  • Git (of course)
  • ImageMagick 6.6+ or later
  • pngnq (optional, for better PNG compression)

How to use

Clone the repository from your command-line. see all avaible colors and sizes for generation. 

$ git clone https://github.com/endel/iconic-css-sprite-sheet.git
$ cd iconic-css-sprite-sheet
$ rake -D

Generate the CSS and PNG Sprite Sheet from your command-line using rake [color]:[size].

$ rake black:12x12 output=my-icons

The output parameter is optional, and should have no extension.

Browser compatibility

Tested on the following browsers. It doesn't mean that it isn't supported by old versions not listed here.

  • Google Chrome 16.0.912.63 beta
  • Safari 5.1.1
  • Firefox 8.0.1
  • Internet Explorer 7+

If you've tested on a older browser please let me know leaving a message on Github.
